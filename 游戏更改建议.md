**文件：** `main.py`

**具体实现：**

1. 在选择调制/编码时，实时计算并显示预估结果：

```python
def estimate_ber(snr_db, modulation, coding):
    # 使用 dsp_engine 的公式计算理论BER
    theoretical_ber = dsp.calculate_theoretical_ber(snr_db, modulation)
    # 根据编码调整
    if coding == "Hamming(7,4)":
        theoretical_ber *= 0.3
    elif coding == "Repetition(3,1)":
        theoretical_ber *= 0.1
    return theoretical_ber
```

1. 在HUD右侧面板显示：

```
│ 预计误码率:         │
```

1. 建议星级算法：
  - 5星：预计BER < 目标BER × 0.5

  - 3星：预计BER < 目标BER
  - 2星：预计BER < 目标BER × 1.5
  - 1星：预计BER > 目标BER × 1.5

**预期效果：** 玩家在发送前就知道配置是否合理

1. 在信号传输动画中增加数据包粒子：

```python
class DataPacket:
    def __init__(self, start_pos, is_correct):
        self.pos = start_pos
        self.color = GREEN if is_correct else RED
        self.progress = 0.0

    def update(self, dt):
        self.progress += dt * 0.5
        # 沿贝塞尔曲线移动
        self.pos = get_bezier_point(self.progress, p0, p1, p2)
```

```python
def calculate_score(ber, target_ber, time_spent, tech_used):
    score = 0

    # 基础分：完成任务
    if ber < target_ber:
        score += 100

    # BER奖励分
    ber_ratio = ber / target_ber
    if ber_ratio < 0.1:
        score += 50
    elif ber_ratio < 0.5:
        score += 30
    elif ber_ratio < 0.8:
        score += 10

    # 速度奖励分
    if time_spent < 15:
        score += 30
    elif time_spent < 30:
        score += 15

    # 技术选择奖励分
    if is_optimal_tech(tech_used, level_id):
        score += 20

    return score
```

### 5.1 智能提示系统（优先级：★★★★☆）

**文件：** `main.py`

**具体实现：**

```python
class HintSystem:
    def __init__(self):
        self.failure_count = {}  # 每关失败次数

    def get_hint(self, level_id, failure_count, last_config):
        if failure_count == 1:
            return "提示: 当前SNR较低，考虑使用纠错码"
        elif failure_count == 2:
            return f"提示: 试试 {get_recommended_coding(level_id)}"
        elif failure_count == 3:
            return f"推荐配置:\n调制: {rec_mod}\n编码: {rec_code}"
        elif failure_count >= 5:
            return "是否使用自动配置？[Y/N]"
        return None
```

在失败界面显示提示，帮助卡关玩家

**预期效果：** 减少挫败感，保持玩家参与度
