# 《大静默》可操作性增强方案

## 🎮 问题分析

**当前玩家操作流程**：
```
选择调制 → 选择编码 → 选择路径 → 点击发送 → 等待结果
```

**问题**：
- ❌ 操作单一，缺乏策略深度
- ❌ 发送后无法干预，只能被动等待
- ❌ 参数固定，无法微调优化
- ❌ 缺乏资源管理和权衡决策
- ❌ 没有实时反馈和动态调整

---

## 💡 增强方案（按优先级排序）

### 🔥 方案1：参数微调系统（优先级：高）

**核心思路**：让玩家可以调节关键参数，看到实时影响

#### 1.1 发射功率控制

```python
# 在HUD中添加功率滑块
class PowerControl:
    def __init__(self):
        self.power_dbm = 30  # 默认30dBm
        self.min_power = 10
        self.max_power = 50
        self.energy_budget = 1000  # 每关的能量预算

    def adjust_power(self, new_power):
        """调整功率，实时显示SNR变化"""
        self.power_dbm = new_power
        # 功率越高，SNR越好，但消耗能量越多
        energy_cost = (new_power - self.min_power) ** 2
        return energy_cost
```

**UI设计**：
```
┌─────────────────────────────────┐
│ 发射功率: [====|====] 35 dBm   │
│           10dB      50dB        │
│                                 │
│ 预期SNR:  12.5 dB  ↑ +2.5      │

│ 剩余能量: ████████░░ 750/1000   │
└─────────────────────────────────┘
```

**玩法**：
- 功率越高，SNR越好，但消耗能量越快
- 玩家需要在"快速高功率"和"节能低功率"之间权衡
- 能量耗尽则任务失败

#### 1.2 带宽调节

```python
class BandwidthControl:
    def __init__(self):
        self.bandwidth_mhz = 1.0  # 默认1MHz
        self.available_bands = [0.5, 1.0, 2.0, 5.0]

    def set_bandwidth(self, bw):
        """带宽越大，速率越高，但噪声功率也越大"""
        self.bandwidth_mhz = bw
        # 噪声功率 = N0 * 带宽
        noise_increase_db = 10 * np.log10(bw / 1.0)
        return noise_increase_db
```

**玩法**：
- 带宽大 → 传输快，但噪声大，BER高
- 带宽小 → 传输慢，但噪声小，BER低
- 玩家需要根据时间限制和质量要求选择

#### 1.3 天线方向调整（激光通信）

```python
class AntennaPointing:
```

### 🎯 方案3：资源管理系统（优先级：中）

**核心思路**：引入资源约束，增加策略深度

#### 3.1 能量预算管理

```python
class EnergyBudget:
    def __init__(self, total_energy):
        self.total = total_energy
        self.remaining = total_energy
        self.consumption_rate = 0

    def calculate_cost(self, power_dbm, duration_sec):
        """计算能量消耗"""
        power_watts = 10 ** ((power_dbm - 30) / 10)
        return power_watts * duration_sec

    def can_afford(self, cost):
        return self.remaining >= cost
```

#### 3.2 时间窗口管理

```python
class TimeWindow:
    def __init__(self, window_start, window_end):
        self.start = window_start
        self.end = window_end
        self.current_time = 0

    def time_remaining(self):
        return self.end - self.current_time

    def is_expired(self):
        return self.current_time >= self.end
```

**场景示例**：

**玩法**：
- 卫星通信有可见时间限制
- 玩家需要计算是否能在窗口内完成
- 可以选择分批传输或提高速率

#### 3.3 频谱资源分配

```python
class SpectrumAllocation:
    def __init__(self):
        self.total_bandwidth = 10  # MHz
        self.allocated = {}

    def allocate(self, link_id, bandwidth):
        """为链路分配带宽"""
        if sum(self.allocated.values()) + bandwidth <= self.total_bandwidth:
            self.allocated[link_id] = bandwidth
            return True
        return False
```

```python
class MultiObjective:
    def __init__(self):
        self.objectives = {
            "ber": {"target": 0.001, "weight": 0.4},
            "time": {"target": 60, "weight": 0.3},
            "energy": {"target": 500, "weight": 0.3}
        }

    def calculate_score(self, ber, time, energy):
        """多目标评分"""
        scores = []
        scores.append(1.0 if ber < 0.001 else 0.5)
        scores.append(1.0 if time < 60 else 0.5)
        scores.append(1.0 if energy < 500 else 0.5)
        return sum(scores) / 3
```
