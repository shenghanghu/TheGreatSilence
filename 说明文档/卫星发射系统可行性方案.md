# 卫星发射系统可行性方案

## 📋 需求分析

### 核心需求
- 玩家可以主动发射卫星作为中继节点
- 部分关卡的中继节点需要玩家自己部署
- 增加策略规划和资源管理维度

### 设计原则（重要）

1. **无槽位设计**：不预设卫星位置，玩家在允许范围内**自行选定轨道/位置**，轨道越远发射预算越高。
2. **发射后位置固定**：卫星发射并部署完成后，**位置固定不变**，不随轨道运动，便于规划路径与链路。
3. **预算优先**：预算作为核心约束，贯穿关卡设计、UI 与策略，优先级高于槽位与数量限制。

### 与现有系统的关系

## 💰 预算设计（高优先级）

预算是卫星发射系统的**首要约束**，优先于槽位、数量等设计，贯穿关卡、UI 与策略。

### 预算原则

- **关卡预算**：每关给出总预算，玩家在预算内选择「发射几颗、放在哪、什么类型」。
- **成本由轨道/距离决定**：玩家自选位置，**距离参考点（如地球/基地）越远，发射成本越高**；可选公式：`成本 = 基础成本(卫星类型) + 距离系数 × 距离`。
- **预算可见、实时反馈**：UI 中预算条、已用/剩余、单次部署预估成本始终可见；选定位置时实时显示「本次发射将消耗 XX」。
- **策略核心**：在预算内权衡「少颗高性能 vs 多颗覆盖」与「近轨省钱 vs 远轨覆盖」。

### 预算与关卡

- 关卡配置中 **budget** 为必填且优先调优项。
- 难度可通过预算松紧调节（紧张预算 = 高难度）。
- 评分/成就可包含「预算节约」「在限定预算内通关」等。

#### 1.2 卫星部署逻辑（无槽位，玩家自定轨道）

- **无预定槽位**：不提供固定槽位列表，玩家在允许的**位置/轨道范围**内自选坐标（如点击或拖拽落点）。
- **成本与距离挂钩**：选定位置后，根据该位置到参考点（如地球/基地）的**距离**计算发射成本，越远成本越高；再叠加卫星类型基础成本。
- **发射后位置固定**：一旦发射确认，该卫星的**位置在整关中固定不变**，不再做轨道运动，仅作为固定中继节点参与路径与 SNR 计算。

```python
class SatelliteDeployment:
    """卫星部署系统（无槽位，玩家自选轨道，预算优先）"""
    def __init__(self, level_config):
        dep = level_config.get("satellite_deployment", {})
        self.budget = dep.get("budget", 1000)
        self.position_range = dep.get("position_range", {"x": (0, 800), "y": (0, 600)})
        self.cost_per_distance = dep.get("cost_per_distance", 2.0)
        self.reference_pos = dep.get("reference_pos", (0, 300))  # 距离计算参考点
        self.deployed_satellites = []

    def get_deploy_cost(self, position, satellite_type):
        """根据玩家选定位置与卫星类型计算成本（越远越贵）"""
        base_cost = SATELLITE_TYPES[satellite_type]["base_cost"]
        distance = np.linalg.norm(np.array(position) - np.array(self.reference_pos))
        return int(base_cost + self.cost_per_distance * distance)

    def can_deploy(self, position, satellite_type):
        """检查位置是否在范围内且预算是否足够"""
        if not self._is_in_range(position):
            return False
        cost = self.get_deploy_cost(position, satellite_type)
        return self.budget >= cost

    def _is_in_range(self, position):
        x_ok = self.position_range["x"][0] <= position[0] <= self.position_range["x"][1]
        y_ok = self.position_range["y"][0] <= position[1] <= self.position_range["y"][1]
        return x_ok and y_ok

    def deploy_satellite(self, position, satellite_type):
        """在玩家选定位置部署卫星（发射后位置固定）"""
        if not self.can_deploy(position, satellite_type):
            return False
        cost = self.get_deploy_cost(position, satellite_type)
        satellite = {
            "name": f"Player Sat {len(self.deployed_satellites) + 1}",
            "pos": tuple(position),
            "type": "relay",
            "specs": self.get_satellite_specs(satellite_type),
            "cost": cost
        }
        self.deployed_satellites.append(satellite)
        self.budget -= cost
        return True
```

#### 2.2 成本由「卫星类型 + 轨道距离」决定（无槽位）

成本 = 卫星类型基础成本 + 距离系数 × 到参考点距离。**越远预算越高**，由玩家自选位置驱动。

```python
def get_satellite_cost(satellite_type, position, reference_pos, cost_per_distance):
    """计算卫星发射成本：基础成本 + 距离成本（越远越贵）"""
    base_cost = SATELLITE_TYPES[satellite_type]["base_cost"]
    distance = np.linalg.norm(np.array(position) - np.array(reference_pos))
    return int(base_cost + cost_per_distance * distance)
```

### 3. 轨道/位置设计（玩家自定，发射后固定）

- **可部署区域**：关卡通过 `position_range`（或等效参数）限定玩家可选的坐标范围，不预设具体槽位。
- **玩家操作**：在区域内点击或拖拽选定落点，即「自定轨道位置」；确认后按该位置计算成本并扣减预算。
- **发射后位置固定**：卫星部署完成后，**位置在整个关卡中保持不变**，不随时间或轨道类型移动，仅作为固定中继节点参与链路与路径计算。

#### 动态更新节点和SNR矩阵

```python
class DynamicNetwork:
    """动态网络管理"""
    def __init__(self, level_config):
        self.base_nodes = level_config["nodes"].copy()
        self.base_snr = np.array(level_config["snr_matrix"]).copy()
        self.deployed_satellites = []

    def add_satellite(self, satellite):
        """添加卫星节点"""
        self.deployed_satellites.append(satellite)

        # 更新节点列表
        self.current_nodes = self.base_nodes + self.deployed_satellites

        # 扩展SNR矩阵
        self.current_snr = self.expand_snr_matrix(satellite)

    def expand_snr_matrix(self, new_satellite):
        """扩展SNR矩阵以包含新卫星"""
        n = len(self.current_nodes) - 1  # 新卫星之前的节点数
        new_size = n + 1

        # 创建新矩阵
        new_snr = np.zeros((new_size, new_size))
        new_snr[:n, :n] = self.current_snr  # 复制旧矩阵

        # 计算新卫星到其他节点的SNR
        for i in range(n):
            snr = self.calculate_snr(
                self.current_nodes[i]["pos"],
                new_satellite["pos"],
                new_satellite["specs"]
            )
            new_snr[i, n] = snr
            new_snr[n, i] = snr

        return new_snr

    def calculate_snr(self, pos1, pos2, satellite_specs):
        """计算两点之间的SNR"""
        distance = np.linalg.norm(np.array(pos1) - np.array(pos2))

        # 简化的链路预算计算
        tx_power = 30  # dBm
        antenna_gain = satellite_specs["antenna_gain"]
        path_loss = 20 * np.log10(distance / 100)  # 简化

        snr = tx_power + antenna_gain - path_loss - 80  # 80dB噪声
        return snr
```

```python
# main.py 中添加新状态
STATE_SATELLITE_DEPLOYMENT = 15  # 新增状态

class GameState:
    def __init__(self):
        self.current_state = STATE_START_SCREEN
        self.satellite_deployment = None
        self.dynamic_network = None

    def enter_satellite_deployment(self, level_config):
        """进入卫星部署阶段"""
        if level_config.get("satellite_deployment", {}).get("enabled", False):
            self.current_state = STATE_SATELLITE_DEPLOYMENT
            self.satellite_deployment = SatelliteDeployment(level_config)
            self.dynamic_network = DynamicNetwork(level_config)
            return True
        return False

    def finish_deployment(self):
        """完成部署，进入游戏阶段（已部署卫星位置固定不变）"""
        min_sats = self.satellite_deployment.level_config.get("min_satellites", 0)
        if len(self.satellite_deployment.deployed_satellites) < min_sats:
            return False, f"至少需要部署{min_sats}颗卫星"

        for sat in self.satellite_deployment.deployed_satellites:
            self.dynamic_network.add_satellite(sat)  # 卫星位置固定，不再改变

        self.current_state = STATE_PLAYING
        return True, "部署完成"
```

### 6.3 评分系统

```python
class SatelliteDeploymentScore:
    """卫星部署评分"""
    def calculate_score(self, deployment):
        """计算部署方案的得分"""
        score = 0

        # 1. 预算节约奖励
        budget_saved = deployment.budget - deployment.total_cost
        score += budget_saved * 0.5

        # 2. 路径质量
        path_snr = deployment.calculate_best_path_snr()
        if path_snr > 15:
            score += 300
        elif path_snr > 10:
            score += 200
        else:
            score += 100

        # 3. 冗余度奖励
        num_paths = deployment.count_viable_paths()
        score += num_paths * 50

        # 4. 效率奖励（少卫星高性能）
        efficiency = path_snr / len(deployment.deployed_satellites)
        score += efficiency * 20

        return int(score)
```

```python
class LaunchAnimation:
    """发射动画"""
    def __init__(self, start_pos, end_pos):
        self.start = start_pos
        self.end = end_pos
        self.progress = 0
        self.duration = 2.0  # 秒

    def update(self, dt):
        """更新动画"""
        self.progress += dt / self.duration
        if self.progress >= 1.0:
            return self.end, True  # 完成

        # 抛物线轨迹
        t = self.progress
        x = self.start[0] + (self.end[0] - self.start[0]) * t
        y = self.start[1] + (self.end[1] - self.start[1]) * t - 100 * np.sin(np.pi * t)
        return (x, y), False
```

```python
# satellite_system.py（新文件）

class SatelliteDeployment:
    def __init__(self, budget, position_range, reference_pos, cost_per_distance=2.0):
        self.budget = budget
        self.position_range = position_range
        self.reference_pos = reference_pos
        self.cost_per_distance = cost_per_distance
        self.deployed = []

    def get_cost(self, position, sat_type):
        base = {"basic": 300, "advanced": 600}[sat_type]
        dist = ((position[0]-self.reference_pos[0])**2 + (position[1]-self.reference_pos[1])**2)**0.5
        return int(base + self.cost_per_distance * dist)

    def _in_range(self, position):
        x, y = position
        rx, ry = self.position_range["x"], self.position_range["y"]
        return rx[0] <= x <= rx[1] and ry[0] <= y <= ry[1]

    def deploy(self, position, sat_type):
        if not self._in_range(position) or self.budget < self.get_cost(position, sat_type):
            return False
        cost = self.get_cost(position, sat_type)
        self.deployed.append({
            "pos": tuple(position),
            "type": "relay",
            "specs": {"antenna_gain": 15 if sat_type == "basic" else 25}
        })
        self.budget -= cost
        return True

    def get_nodes(self):
        return self.deployed  # 发射后位置固定
```
