# 《大静默》成就系统设计

## 一、成就系统概述

成就系统旨在：
- 引导玩家探索不同技术组合
- 鼓励玩家追求更高表现
- 增加游戏重玩价值
- 提供长期游戏目标

### 3.2 成就检测系统

```python
# achievements.py (续)

class AchievementManager:
    def __init__(self):
        self.unlocked = set()
        self.newly_unlocked = []

    def check_achievements(self, game_stats):
        """检测成就解锁"""
        self.newly_unlocked = []
        for ach_id, ach_data in ACHIEVEMENTS.items():
            if ach_id not in self.unlocked:
                if ach_data["check"](game_stats):
                    self.unlocked.add(ach_id)
                    self.newly_unlocked.append(ach_id)
        return self.newly_unlocked

    def get_achievement(self, ach_id):
        return ACHIEVEMENTS.get(ach_id)

    def get_progress(self):
        total = len(ACHIEVEMENTS)
        unlocked = len(self.unlocked)
        return unlocked, total

    def get_by_category(self, category):
        return {k: v for k, v in ACHIEVEMENTS.items() if v["category"] == category}

    def save(self):
        return list(self.unlocked)

    def load(self, data):
        self.unlocked = set(data)
```

### 3.4 成就解锁触发时机

```python
# 在 main.py 中的关键位置调用成就检测

# 1. 关卡完成时
def on_level_complete(level_id, ber, stars, time_spent, score, modulation, coding):
    # 更新统计数据
    game_stats["levels_completed"] += 1
    game_stats["level_stars"][level_id] = max(game_stats["level_stars"].get(level_id, 0), stars)
    game_stats["best_ber"] = min(game_stats["best_ber"], ber)
    game_stats["fastest_time"] = min(game_stats["fastest_time"], time_spent)
    game_stats["highest_score"] = max(game_stats["highest_score"], score)
    game_stats[f"level_{level_id}_score"] = max(game_stats.get(f"level_{level_id}_score", 0), score)

    # 更新技术使用统计
    game_stats[f"{modulation.lower()}_clears"] += 1
    game_stats[f"{coding.lower()}_clears"] += 1

    # 更新连胜记录
    if stars >= 3:
        game_stats["consecutive_three_stars"] += 1
    else:
        game_stats["consecutive_three_stars"] = 0

    # 检测成就
    newly_unlocked = achievement_manager.check_achievements(game_stats)
    if newly_unlocked:
        show_achievement_popup(newly_unlocked)

# 2. 发送信号时
def on_signal_sent():
    game_stats["total_transmissions"] += 1
    achievement_manager.check_achievements(game_stats)

def game_loop():
```

```python
    json.dump(data, open("save_data.json", "w"))
```

### 4.3 关键事件钩子

```python
# 关卡完成
def on_level_complete(level_id, result):
    update_stats_on_complete(level_id, result)
    newly_unlocked = achievement_manager.check_achievements(game_stats)
    if newly_unlocked:
        queue_achievement_popups(newly_unlocked)

# 技术解锁
def unlock_technology(tech_name):
    game_stats["unlocked_techs"].append(tech_name)
    achievement_manager.check_achievements(game_stats)

# 配置尝试
def try_configuration(level_id, modulation, coding):
    config_key = f"{level_id}_{modulation}_{coding}"
    game_stats["tried_combinations"].add(config_key)

    level_configs = game_stats.get(f"level_{level_id}_configs", set())
    level_configs.add(f"{modulation}_{coding}")
    game_stats[f"level_{level_id}_configs"] = level_configs
    game_stats["max_configs_tried"] = max(
        game_stats["max_configs_tried"],
        len(level_configs)
    )
```

## 六、注意事项

1. **性能考虑**：成就检测只在关键事件后执行，避免每帧检测
2. **数据持久化**：及时保存成就解锁状态，防止数据丢失
3. **平衡性**：成就难度有梯度，确保大部分玩家能解锁40-60%
4. **用户体验**：成就解锁要有明显反馈，不打断游戏流程
5. **可维护性**：成就定义与检测逻辑分离，便于调整
